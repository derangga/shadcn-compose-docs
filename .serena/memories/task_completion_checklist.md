# Task Completion Checklist

When a coding task is completed, verify the following:

1. **Build check**: Run `bun run build` to ensure no TypeScript or build errors
   - This is the ONLY verification step available (no linter, formatter, or tests configured)

2. **Route tree**: If routes were added/modified, the route tree (`src/routeTree.gen.ts`) is auto-generated by TanStack Start — do NOT edit it manually. It regenerates on dev server start or build.

3. **Component menu**: If a new component doc page was added, ensure it's registered in:
   - `src/lib/component-menu.ts` (sidebar menu)
   - `src/types/content.ts` (Component union type)

4. **MDX components**: If new MDX-embeddable components were created, register them in `src/components/mdx/mdx-components.tsx`

5. **Import style**: Ensure `import type` is used for type-only imports (enforced by `verbatimModuleSyntax`)
