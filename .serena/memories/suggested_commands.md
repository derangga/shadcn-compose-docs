# Suggested Commands

## Package Management (Bun)
- `bun install` — Install dependencies
- `bun add <package>` — Add a dependency
- `bun add -d <package>` — Add a dev dependency

## Development
- `bun run dev` — Start Vite dev server (TanStack Start SSR)
- `bun run build` — Production build (only verification step)
- `bun run preview` — Preview production build via Vite preview

## Deployment
- `bun run deploy` — Build + deploy to Cloudflare Workers via wrangler

## System Utilities (macOS/Darwin)
- `git` — Version control
- `ls`, `cd`, `find`, `grep` — Standard unix utilities
- `cat`, `head`, `tail` — File viewing
